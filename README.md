# SQL End-to-End App

### Tables Included:
- **Customer**: id, name, email
- **Loan**: id, amount, status
- **Transition**: id, status, date
- **Charge**: id, desc, cost
- **Orders**: id, product
- **Failure**: id, reason
- **Cancellation**: id, reason

### How to use:
1. Open `index.html` in your browser.
2. Write SQL queries in the editor.
3. Click "Run Query" to see data.
